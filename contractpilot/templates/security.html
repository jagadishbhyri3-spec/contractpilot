Security

We take security seriously. ContractPilot is built with enterprise-grade protections to keep your contracts and data safe.

TLS 1.3 Encryption
SOC 2 Aligned
Password Hashing (bcrypt)
EU Data Hosting

Data Encryption

All data in transit is protected with TLS 1.3. Data at rest is encrypted using AES-256. Contract documents are stored in isolated storage with strict access controls.

Authentication & Access

  • Passwords hashed with bcrypt (cost factor 12+)
  • JWT tokens with secure, httpOnly cookie options
  • Session expiration and automatic logout
  • No plaintext password storage ever

Infrastructure

Hosted on Render with data centers in the EU (Frankfurt). Our infrastructure provider maintains ISO 27001, SOC 2 Type II, and GDPR compliance certifications.

AI Data Handling

Contract text is transmitted to Google AI Studio (Gemini) for analysis. We do not use your contract data to train AI models. Google processes data under their enterprise terms with data protection guarantees. Contract data is not retained by Google beyond the processing session.

Incident Response

We maintain a documented incident response plan. In the event of a data breach, we will notify affected users within 72 hours as required by GDPR. Contact security@contractpilot.co to report vulnerabilities.

Compliance Roadmap

  • ✅ GDPR / UK GDPR compliant
  • ✅ SOC 2 Type I controls implemented
  • 🔄 SOC 2 Type II audit scheduled Q4 2026
  • 🔄 ISO 27001 certification planned 2027

Contact Security

Report security issues to security@contractpilot.co. We operate a responsible disclosure program.
