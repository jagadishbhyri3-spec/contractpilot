Security

We take security seriously. Here's how we protect your data.

Encryption

All data in transit uses TLS 1.3 encryption. Passwords are hashed using bcrypt with adaptive cost factors.

Data Hosting

All data is stored in Frankfurt, Germany, within the EU. We are fully GDPR compliant.

No AI Training

We never use your contract data to train AI models. Your documents are processed for analysis only and remain confidential.

Report a Vulnerability

Found a security issue? Contact security@contractpilot.co.
